# Meetings & Scheduling System

## Version 1 Scope

### Supported Providers

* Zoom Links
* Google Meet Links
* Microsoft Teams Links
* Custom Meeting Links

### Features

* Create Meeting Session
* Assign Teacher
* Assign Students
* Set Date & Time
* Store Meeting Link
* Edit Meeting Link
* Replace Meeting Link
* Cancel Meeting
* Upcoming Meetings
* Previous Meetings
* Student Join Button
* Teacher Join Button

### Notifications

* 24 Hours Before Session
* 1 Hour Before Session
* 15 Minutes Before Session

### Student Experience

Student sees:

* Session Title
* Teacher
* Date
* Time
* Join Button

### Teacher Experience

Teacher can:

* Create Session
* Edit Session
* Delete Session
* Change Meeting Link
* View Students

### Admin Experience

Admin can:

* Manage All Sessions
* Manage All Meeting Links
* Override Teacher Sessions
* Assign Sessions

## Out Of Scope (Version 1)

The following are NOT required:

* Zoom API Integration
* Google Meet API Integration
* Auto-Creation of Meetings
* Built-In Video Calling
* Live Streaming Infrastructure

Version 1 only stores and manages external meeting links.
